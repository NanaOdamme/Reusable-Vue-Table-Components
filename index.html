Reusable Vue 3 Table Component

Documentation and Integration Guide

Uses Vue 3, Tailwind CSS, and Bootstrap Icons via CDN

A reusable, dynamic, and theme-able Vue 3 data table component that supports custom headers, full CRUD (Create, Read, Update, Delete) operations, and both local and API-driven data handling.

Installation:

Include the component in your Vue 3 project:

Include the following CDN links in your HTML file:

            

            
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script src="https://cdn.tailwindcss.com"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons/font/bootstrap-icons.css">

Then, import the component in your Vue file:

document.addEventListener("DOMContentLoaded", function () {
    const app = Vue.createApp({
        components: {
            TableComponent
        },
        
        ....});
    app.mount("#app");});

Key Features

  • Dynamic headers (custom fields and keys) and CRUD control
  • Supports API and local data
  • Theme support
  • Fully customizable logic
  • Emits clean payloads for integration into any project

Usage

<TableComponent
  :headers="tableHeaders"
  :data="tableData"
  :crud-options="crudOptions"
  :selected-theme="selectedTheme"
  @add="handleAdd"
  @edit="handleEdit"
  @delete="handleDelete"
/>

Props

  • headers — array of column definitions
  • tableData — array of row data
  • crudOptions — enable/disable Add/Edit/Delete
  • selectedTheme — theme for the table (default, dark, light, etc.)

Header Format


        tableHeaders = [
          { label: "Display Name", // e.g., "Country"
            key: "propertyName",   // e.g., "name"
            // The key used to access the data in each row
            type: "text" | "number" | "date" | "select - options[value1, value2, ...]" | "image",]",
            display: true | false  // Whether to show this column
        }
        ];

Example:

[
  { label: "Name", key: "name", type: "text", display: true },
  { label: "Email", key: "email", type: "text", display: true }
]

CRUD Options

Set Which actions are allowed:


        crudOptions: {
                    allowAdding: true, //or false
                    allowEditing: true, //or false
                    allowDeleting: true //or false
                    }

Events

Emitted events for CRUD operations:

  • @add — Emitted when a new item is added
  • @edit — Emitted when an item is edited
  • @delete — Emitted when items are deleted

CRUD Handlers

These methods must be defined in your component. They are completely customizable, allowing API or local data handling.

handleAdd(newItem)

Parameters:

Name Type Description
newItem Object The new row object from form

Example (with API):

async handleAdd(newItem) {
  const response = await axios.post('/api/endpoint', newItem);
  this.tableData.push(response.data);
}

Example (with local data):

handleAdd(newItem) {
            newItem.id = Date.now(); // Generate a unique ID
            this.tableData.push(newItem);
        }

handleEdit(updatedItem, index)

Parameters:

Name Type Description
updatedItem Object The updated row object from form
index Number The index of the item in the tableData array

Example (with API):

async handleEdit(updatedItem, index) {
        const payload = {...updatedItem, Id: updatedItem.id}; // Ensure the ID is included
        delete payload.id; // Remove id if not needed in the payload

  // This is to ensure the API receives the correct data structure
  // Assuming the API expects the ID to update the correct item

  await axios.put(`/api/endpoint/${updatedItem.id}`, payload);
  this.tableData.splice(index, 1, updatedItem);
}

Example (with local data):

handleEdit(updatedItem, index) {
    this.tableData.splice(index, 1, updatedItem);
}

Note: The index is provided to help you update the correct item in the tableData array.

handleDelete(deletedItems)

Parameters:

Name Type Description
deletedItems Array An array of items to be deleted

Example (with API):

async handleDelete(deletedItems) {
  for (const item of deletedItems) {
    await axios.delete(`/api/endpoint/${item.id}`);
  }
  this.tableData = this.tableData.filter(row => !deletedItems.includes(row));
}

Example (with local data):

handleDelete(deletedItems) {
    this.tableData = this.tableData.filter(row => !deletedItems.includes(row));
}

Theme support

To change the theme, set the selectedTheme data property to one of the available themes:

Example:

data() {
    return {
        selectedTheme: "default" // or "dark", "light", etc.
        //there are 20 themes to select from

    };
    }

Example Integration

In your Js

document.addEventListener("DOMContentLoaded", function () {
    const app = Vue.createApp({
        components: {
            TableComponent
        },
        data() {
            return {
                tableHeaders: [
                    { label: "Country", type: "text", key: "name", display: true },
                    { label: "Nationality", type: "text", key: "nationality", display: true },
                    { label: "Country Code", type: "text", key: "code", display: true }
                ],
                tableData: [],
                crudOptions: {
                    allowAdding: true,
                    allowEditing: true,
                    allowDeleting: true
                },
                selectedTheme: "default",
                dataList: [],
            };
        },
        methods: {
            async getTableData() {
                try {
                    const response = await axios.get("/setups/api/nationalities");
                    this.dataList = response.data;
                    this.dataList.sort((a, b) => a.name.localeCompare(b.name));
                    this.tableData = this.dataList;
                } catch (error) {
                    console.error(error);
                    toastr.error(error.message || "Error loading data");
                }
            },

            async handleAdd(newItem) {
                try {
                    const response = await axios.post("/setups/api/nationalities", newItem);
                    this.tableData.push(response.data);
                    toastr.success("Nationality added successfully");
                } catch (error) {
                    console.error(error);
                    toastr.error(error.message || "Error adding nationality");
                }
            },

            async handleEdit(updatedItem, index) {
                try {
                    const payload = { ...updatedItem, Id: updatedItem.id };
                    delete payload.id;

                    await axios.put(`/setups/api/nationalities/${updatedItem.id}`, payload);
                    this.tableData.splice(index, 1, updatedItem);
                    toastr.success("Nationality updated successfully");
                } catch (error) {
                    console.error(error);
                    toastr.error(error.message || "Error updating nationality");
                }
            },

            async handleDelete(deletedItems) {
                try {
                    for (const item of deletedItems) {
                        await axios.delete(`/setups/api/nationalities/${item.id}`);
                        this.tableData = this.tableData.filter(c => c.id !== item.id);
                    }
                    toastr.success("Nationality(s) deleted successfully");
                } catch (error) {
                    console.error(error);
                    toastr.error(error.message || "Error deleting nationality");
                }
            }
        },
        mounted() {
            this.getTableData();
        }
    });

    app.mount("#app");
});

In your template (html, cshtml, etc)

<div id="app">
    <TableComponent
        :headers="tableHeaders"
        :data="tableData"
        :crud-options="crudOptions"
        :selected-theme="selectedTheme"
        @add="handleAdd"
        @edit="handleEdit"
        @delete="handleDelete"
    />
</div>
